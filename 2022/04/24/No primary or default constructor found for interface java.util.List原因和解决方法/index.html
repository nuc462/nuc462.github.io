No primary or default constructor found for interface java.util.List原因和解决方法

发表于 2022-04-24 分类于 异常

原因

是list是接口，没有构造函数

解决方法：

  • 换成ArrayList就可以，
  • 将List转为Json字符串（JSON.toJSONString(list)），或者加入@RequestBody注解
  • 目标对象不匹配，用＠RequestParam让他强制匹配
